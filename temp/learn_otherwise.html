Otherwise in SanCode

otherwise is used as a fallback when a condition is false. It works together with when.

Syntax

when condition say value otherwise say value

Example

x 5 when x greater 10 say "Big" otherwise say "Small"

Explanation

If x is greater than 10, it prints Big. If not, the otherwise part runs and prints Small.

Output

Small
