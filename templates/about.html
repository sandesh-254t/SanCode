About SanCode

SanCode is an experimental programming language built with simplicity and clarity in mind.

Vision

Make programming feel natural, readable, and beginner-friendly.

Mission

Remove unnecessary symbols and complexity. Replace them with logic that feels like plain English.

Built By

SanCode was created by Sandesh as a step toward simplifying how people learn programming.

Future Goals

Expand SanCode into a fully functional ecosystem with IDE tools, learning systems, and cloud execution.
